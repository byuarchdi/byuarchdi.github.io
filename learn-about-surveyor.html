Surveyor

Surveyor allows users to collect pedestrian survey data in a paperless format. Data entered in all fields are collected in a database located on the tablet itself. Currently, all entry fields are based on the new Intermountain Antiquities Computer System (IMACS) forms. Data can be shared between mobile and desktop/laptop devices by plugging in a mobile device, running Surveyor, into a desktop or laptop computer with a USB cable. Eventually data transfers may occur via Wi-Fi, Bluetooth, or NFC.
